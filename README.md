# TheMovieNetwork

## Launch server

    export FLASK_APP=project
    export FLASK_DEBUG=1
    flask run --host=0.0.0 --port=1234

## Dependencies
* Python
* MySQL
* Neo4j
* Flask
* Open source [bootstrap template](https://themewagon.com/themes/free-bootstrap-4-html-5-admin-dashboard-template-material-able/)
